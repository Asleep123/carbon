RequestClientOptions

RequestClientOptions: object

The options used to initialize the RequestClient

Type declaration

NameTypeDescription
apiVersion?numberThe version of the API to use. Default 10
baseUrl?stringThe base URL of the API. Default https://discord.com/api
queueRequests?booleanWhether or not to queue requests if you are rate limited. If this is true, requests will be queued and wait for the ratelimit to clear. If this is false, requests will be made immediately and will throw a RateLimitError if you are rate limited. Default true
timeout?numberThe timeout for requests. Default 15000
tokenHeader?"Bot" | "Bearer"The header used to send the token in the request. This should generally always be "Bot" unless you are working with OAuth. Default "Bot"
userAgent?stringThe user agent to use when making requests. Default DiscordBot (https://github.com/buape/carbon, v0.0.0)
